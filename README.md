# Claude Code Skills

Drop any folder into `~/.claude/skills/` to install.

## Skills

| Skill | Description |
|-------|-------------|
| **deep-research** | Comprehensive research reports with citations, powered by MindStudio |
| **remy** | Remy by MindStudio - spec-first AI app builder |
| **mindstudio-expert** | Building, editing, and debugging MindStudio AI agents |

## Install

```bash
# Clone the whole collection
git clone https://github.com/grantmatto/claude-skills

# Copy skills you want
cp -r claude-skills/deep-research ~/.claude/skills/
cp -r claude-skills/remy ~/.claude/skills/
cp -r claude-skills/mindstudio-expert ~/.claude/skills/
```
